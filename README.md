# Stylized-scene-reconsrutcion-through-DDS-of-3DGS
Artistic stylization of 3D Gaussian Splatting via single-step diffusion. Combines Delta Distillation Sampling (DDS) with Difix for artifact-free style transfer, ControlNet for structural preservation, IP-Adapter for style guidance, and MVFC for multi-view consistency. Runs on a single consumer GPU.
